# GigConnect - Campus Gig Marketplace

A platform connecting students with local gig opportunities.

## Features
- Student dashboard to browse and apply for gigs
- Provider dashboard to post and manage gigs
- Real-time application tracking
- Payment management
- Review system

## Tech Stack
- React + Vite
- Tailwind CSS
- Lucide Icons

## Setup
1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`
4. Open http://localhost:5173

## Database Schema
The app uses a 6-table relational database:
- Users (authentication)
- Students (student profiles)
- Providers (business profiles)
- Gigs (job postings)
- Applications (student applications)
- Transactions (payment records)
- Reviews (rating system)
